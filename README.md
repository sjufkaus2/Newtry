## About
더 편하고 모바일 친화적인 엔트리 클라이언트입니다.

## 구현 방법
Newtry는 엔트리 API를 사용하여 작품, 게시글, 유저 정보를 불러옵니다.  
API는 https://api.thoratica.net 에 정리되어 있습니다.
